## 🔧 MCP 工具说明

### 1. `self.pet.list_types` - 列出所有宠物类型

**描述**：列出所有可用的宠物类型，按分类组织。

**参数**：无

**返回示例**：
```json
{
  "domestic": [
    {
      "id": "cat",
      "name": "猫咪",
      "name_en": "Cat",
      "emoji": "🐱",
      "category": "domestic",
      "hunger_rate": 0.8,
      "clean_rate": 1.2,
      "mood_decay_rate": 0.9,
      "personality": "独立、优雅、偶尔高冷",
      "special_trait": "夜间活跃度+20%"
    }
    // ... 更多家养宠物
  ],
  "wild": [
    // ... 野生动物
  ],
  "exotic": [
    // ... 奇异宠物
  ],
  "current": "cat"
}
```

---

### 2. `self.pet.select_type` - 选择宠物类型

**描述**：切换当前的宠物类型。

**参数**：
- `type_id` (string, 必需): 宠物类型 ID（如 "cat", "panda", "dragon" 等）

**示例**：
```json
{"name": "self.pet.select_type", "arguments": {"type_id": "panda"}}
```

**效果**：
- 立即切换宠物类型
- 保留当前的心情、饱腹度、清洁度等状态
- 应用新宠物的衰减倍率

---

### 3. `self.pet.get_type_info` - 获取宠物类型详情

**描述**：获取特定宠物类型的详细信息。

## 📊 宠物属性说明

### 衰减速度倍率

| 倍率 | 含义 | 示例 |
|------|------|------|
| **0.5x** | 超慢 | 树懒（0.5x 饥饿速度） |
| **0.8x** | 较慢 | 猫咪（0.8x 饥饿速度） |
| **1.0x** | 正常 | 基准速度 |
| **1.5x** | 较快 | 小狗（1.5x 脏污速度） |
| **2.0x** | 很快 | 熊猫（2.0x 饥饿速度）、犀牛（2.0x 脏污速度） |
| **2.5x** | 极快 | 大象（2.5x 饥饿速度） |

### 实际衰减速度计算

以熊猫🐼为例：
- 基础饥饿速度：1/分钟
- 熊猫倍率：2.0x
- 实际衰减：2/分钟（每分钟饱腹度 -2）

以树懒🦥为例：
- 基础饥饿速度：1/分钟
- 树懒倍率：0.5x
- 实际衰减：0.5/分钟（每2分钟饱腹度 -1）

## 🔍 常见问题

### Q1: 切换宠物后，之前的状态会丢失吗？
**A**: 不会！切换宠物只会改变衰减倍率和特殊能力，你的心情、饱腹度、清洁度、每日任务进度等都会保留。

### Q2: 可以随时切换宠物吗？
**A**: 可以！随时调用 `self.pet.select_type` 即可切换，没有冷却时间。

### Q3: 哪种宠物最好养？
**A**: 🦥 树懒！所有衰减速度只有0.4-0.6x，是最省心的宠物。其次是🐨 考拉和🦄 独角兽。

### Q4: 哪种宠物最难养？
**A**: 🐘 大象！饥饿速度2.5x，需要频繁喂食。其次是🐼 熊猫（2.0x）和🦏 犀牛（脏污2.0x）。

### Q5: 特殊能力会生效吗？
**A**: 目前特殊能力主要通过衰减倍率体现。未来版本会实现更多特殊能力效果。

### Q6: 可以自定义新的宠物类型吗？
**A**: 可以！修改 `main/pet_system.cc` 中的 `InitializePetTypes()` 函数，添加你自己的宠物类型。

## 📝 开发者：如何添加新宠物

在 `main/pet_system.cc` 的 `InitializePetTypes()` 中添加：

```cpp
pet_types_["your_pet_id"] = {
    "宠物名称",           // 中文名
    "Pet Name",          // 英文名
    "🐾",                // Emoji
    "category",          // 分类：domestic/wild/exotic
    1.0f,                // 饥饿速度倍率
    1.0f,                // 脏污速度倍率
    1.0f,                // 心情衰减倍率
    "性格描述",          // 性格
    "特殊能力描述",      // 特殊能力
    true                 // 是否可用
};
```
