# 小智 AI 机器人 - 用户使用说明

## 👋 欢迎使用小智 AI！

小智是一款智能语音机器人，可以和你聊天、回答问题、讲故事、控制智能家居，还能陪你玩耍！

**固件版本**：v2.0.3  
**更新日期**：2025-10-23  
**核心芯片**：ESP32-S3

---

## 🎯 小智能做什么？

### 💬 和你聊天
- 说"你好小智"就能唤醒我
- 和我聊天，我会回答你的问题
- 我能记住聊天内容，像朋友一样对话

### 🎵 娱乐功能
- 讲笑话、讲故事
- 唱歌、背诗
- 猜谜语、脑筋急转弯
- 陪你玩成语接龙

### 🏠 智能控制
- 控制家里的智能灯、空调、电视
- 查询天气、时间、日期
- 设置闹钟和提醒
- 控制音量和屏幕亮度

### 🐾 虚拟宠物
- 照顾虚拟小宠物
- 给宠物喂食、洗澡、玩耍
- 查看宠物状态和心情
- 完成每日任务获得奖励

### 🎨 表情互动
- 根据心情显示不同表情
- 开心、难过、思考、惊讶
- 表情会随着对话内容变化

## 🚀 快速开始

### 第一步：开机
1. 长按电源键 3 秒开机
2. 等待屏幕显示笑脸表情
3. 听到"欢迎使用小智 AI"的提示音

### 第二步：连接 WiFi

**首次使用需要连接 WiFi：**

1. 小智会自动进入配网模式（显示 WiFi 图标）
2. 用手机连接小智的 WiFi 热点（名称：`XiaoZhi-XXXXXX`）
3. 打开手机浏览器，会自动跳转到配置页面
4. 选择你家的 WiFi 名称，输入密码
5. 点击"连接"，等待小智连接成功
6. 小智会自动重启，连接到你家的 WiFi

**提示**：如果没有自动跳转，可以手动访问：`http://192.168.4.1`

### 第三步：开始使用

连接成功后，对着小智说：

```
"你好小智"
```

小智会回应你，然后就可以开始对话啦！

## 🗣️ 如何和小智对话

### 唤醒词

**说"你好小智"唤醒**，小智会回应：
- "在呢~"
- "有什么事吗？"
- "我在听！"

### 对话技巧

**✅ 正确的说话方式：**
- 等小智说完"叮"的提示音后再说话
- 说话清晰，不要太快或太慢
- 每次说话保持在 1-30 秒
- 安静的环境下识别效果最好

**❌ 避免这样：**
- 提示音还没响就开始说话
- 离太远（超过 3 米）
- 环境太吵（音乐、电视声音太大）
- 说话含糊不清

## 🔋 充电和电量管理

### 充电方式

1. 使用 USB-C 充电线连接
2. 推荐使用 5V/2A 充电器
3. 充电时屏幕会显示充电图标
4. 充满电约需 2-3 小时

### 电量提醒

- **电量 < 20%**：小智会提醒"电量不足，请充电"
- **电量 < 10%**：屏幕显示红色电池图标
- **电量 < 5%**：自动关机保护

### 续航时间

- **正常使用**：6-8 小时
- **待机模式**：24-48 小时
- **连续对话**：4-6 小时

## 💡 使用技巧

### 提高识别准确率

**1. 说话技巧**
- 清晰地说出唤醒词"你好小智"
- 等待提示音后再说话
- 与小智保持 0.5-2 米距离
- 说话音量适中，不要太大声或太小声

**2. 环境优化**
- 在安静的环境中使用
- 关闭电视、音乐等背景噪音
- 避免多人同时说话
- 不要对着风扇或空调说话

**3. 唤醒词技巧**
- 如果第一次没唤醒，稍等 1 秒再说
- 说"你-好-小-智"（稍微慢一点）
- 音调保持平稳，不要忽高忽低

### 打断功能（Barge-in）

**小智正在说话时，你可以随时打断：**
```
小智：我来给你讲一个很长很长的故事...
你：你好小智（打断）
小智：（立即停止）在呢！
```

### 连续对话

唤醒后，可以连续说多句话，不需要每次都说唤醒词：
```
你：你好小智
小智：在呢~

你：今天天气怎么样？
小智：今天晴天...

你：那明天呢？
小智：明天多云...

你：谢谢
小智：不客气！
```

> **注意**：如果超过 10 秒没有说话，需要重新唤醒

### 宠物养成技巧

**1. 保持健康状态**
- 每天至少喂食 2 次
- 每天至少洗澡 1 次
- 经常互动，保持心情愉快

**2. 完成每日任务**
- 快速完成：喂食 → 洗澡 → 玩耍 → 聊天 5 次
- 大约需要 5 分钟
- 连续登录 3 天有额外奖励

**3. 冷却时间**
- 喂食：60 秒
- 洗澡：120 秒
- 玩耍：60 秒
- 拥抱：30 秒

利用冷却时间和小智聊天吧！

## 🆘 常见问题

### Q1: 小智不响应唤醒词怎么办？

**可能原因：**
- 环境太吵，麦克风听不清
- 距离太远（超过 3 米）
- 说话声音太小

**解决方法：**
1. 靠近一点，清晰地说"你好小智"
2. 降低环境噪音（关电视/音乐）
3. 说话声音稍大一点
4. 检查麦克风是否被遮挡

### Q2: 识别不准确，回答牛头不对马嘴？

**可能原因：**
- 网络延迟或不稳定
- 说话不清晰
- 背景噪音干扰

**解决方法：**
1. 检查 WiFi 信号（屏幕左上角图标）
2. 说话清晰，语速适中
3. 在安静环境下使用
4. 重启小智试试

### Q3: 小智说话有回声或噪音？

**解决方法：**
1. 调低音量
2. 远离墙壁和硬质表面
3. 检查扬声器是否被遮挡
4. 重启设备

### Q4: 连不上 WiFi？

**检查步骤：**
1. 确认 WiFi 密码正确
2. 确认路由器正常工作
3. 确认小智在 WiFi 信号范围内
4. 尝试重启路由器
5. 尝试重新配网（长按配置键 5 秒）

**支持的 WiFi：**
- 频段：2.4GHz（大部分型号）或 5GHz（部分型号）
- 加密：WPA2/WPA3
- 不支持企业级 WiFi（需要用户名密码的）

### Q5: 电量消耗很快？

**节省电量方法：**
1. 降低屏幕亮度
2. 降低音量
3. 减少唤醒次数
4. 不用时说"休眠"进入省电模式

### Q6: 宠物状态突然变差了？

**这是正常的！**
- 宠物状态会自动下降（模拟真实宠物）
- 每分钟：饱腹度 -1，清洁度 -1
- 10 分钟无互动：心情额外下降

**解决方法**：定期照顾宠物，完成每日任务

### Q7: 如何升级固件？

**方法 1：自动升级（推荐）**
- 连接 WiFi 后，小智会自动检查更新
- 如果有新版本，会提示"发现新版本，是否升级？"
- 说"升级"确认

**方法 2：手动升级**
- 说"你好小智"
- 说"检查更新"
- 按提示操作

**方法 3：电脑升级**
- 访问官网下载最新固件
- 使用烧录工具升级（需要技术知识）

### Q8: 忘记 WiFi 密码，如何重置？

**恢复出厂设置：**
1. 找到重置按钮（通常在底部或侧面）
2. 用牙签按住重置键 10 秒
3. 小智会恢复出厂设置
4. 重新配置 WiFi

> ⚠️ **警告**：恢复出厂设置会清除所有数据（包括宠物状态）
